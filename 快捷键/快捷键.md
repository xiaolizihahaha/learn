# 快捷键

- Shift command space		#搜索
- command tab					#切程序
- option space					#剪贴板
- command q						#退出
- 选择word + space				#预览
- command 上下左右			#快速归位
- option 左右							#切单词
- command + shift + 。            #隐藏文件显示
- 缩放 control + 双指上下手势



### 1.命令行快捷键

- control + a			#快速到行首
- control + e            #快速到行尾
- control + c             #快速删除当前指令





### 2.vim快捷键

- dd                           #vim删除一行
- shift + g               #vim到文本末尾
- g + g                       #vim到文本首
- shift + 4                 #vim到该行最后一个字符
